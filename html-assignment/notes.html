"img" tag
- allows you to add images to a page, with original sizing (be sure to source wit with "src" and add the "alt" attribute for a description)
"ol,ul and dl" tags
different tags for different list types. ol is for lists with an order, ul is lists that don't have order important, and you are currently reading a descriptive list right now.
"a" tag
Anchor tag, used for linking to a site. Can be opened on the same page or a diffferent one.... depending on attributes.
"h1,h2,h3...." tags
heading tags, the size varies by the number. The higher the number, sthe smaller the size.
"footer"tag
